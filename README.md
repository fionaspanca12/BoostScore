# BoostScore
BoostScore

BoostScore is a app that offers esports live scores and results.

# Features 

 * **LayoutManagers**
 * **RecyclerView**
 * **SplashActivity**
 * **Fragments**
 * **RecyclerView.Adapter**  
 * **SharedPreferences**  
 * **SQLiteDatabase**  
 * **MenuItem**  
 * **Toolbar**
